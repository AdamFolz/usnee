# Установка USNEE на Windows (без Docker)

Это пошаговая инструкция для запуска проекта локально на компьютере. Нужно будет скачать 2 программы и запустить 3 команды.

---

## Шаг 1. Установить Node.js (для frontend)

1. Открой браузер и перейди на сайт: **https://nodejs.org**
2. Нажми большую зелёную кнопку **"LTS"** (длинный номер версии, например 20.x.x)
3. Скачай установщик и запусти его
4. Жми **"Next"** пока не закончится установка (ничего менять не нужно)
5. **Перезагрузи компьютер** после установки

Проверка: открой **Командную строку** (нажми Win+R, напиши `cmd`, нажми Enter) и введи:
```
node --version
```
Если вышло что-то вроде `v20.x.x` — всё ок.

---

## Шаг 2. Установить Python (для backend)

1. Открой браузер и перейди на сайт: **https://python.org/downloads**
2. Нажми жёлтую кнопку **"Download Python 3.11.x"**
3. Запусти скачанный файл
4. **ВАЖНО:** на первом экране поставь галочку **"Add Python to PATH"** (внизу окна)
5. Жми **"Install Now"** и дождись окончания
6. **Перезагрузи компьютер**

Проверка: в Командной строке введи:
```
python --version
```
Если вышло `Python 3.11.x` — всё ок.

---

## Шаг 3. Настроить файл .env (настройки)

Этот файл уже создан в папке проекта. Он называется `.env` и лежит в `M:\SEKS\АКТУАЛЬНОЕ\USNEE\`.

Открой его **Блокнотом** (Notepad). Там уже есть тестовые настройки. Можешь ничего не менять, но если хочешь:

- `BOT_TOKEN` — токен от Telegram-бота (получить у @BotFather). Для локального теста можно оставить как есть.
- `DEV_MODE=true` — это включено, чтобы можно было тестировать без Telegram
- `DEV_USER_ID=123456789` — тестовый ID пользователя

**Сохрани файл** (Ctrl+S) и закрой.

---

## Шаг 4. Запустить backend (Python)

1. Открой **Командную строку** (Win+R → `cmd` → Enter)
2. Перейди в папку backend:
```
cd M:\SEKS\АКТУАЛЬНОЕ\USNEE\backend
```
3. Создай виртуальное окружение:
```
python -m venv venv
```
4. Активируй его:
```
venv\Scripts\activate
```
(В начале строки должно появиться `(venv)`)

5. Установи зависимости:
```
pip install -r requirements.txt
```
(Жди, пока скачается всё — это может занять 2-3 минуты)

6. Запусти backend:
```
uvicorn app.main:app --reload --port 8000
```

Если всё ок, ты увидишь строчки:
```
INFO:     Uvicorn running on http://127.0.0.1:8000
INFO:     Application startup complete.
```

**Оставь это окно открытым** — backend работает.

---

## Шаг 5. Запустить frontend (React)

1. Открой **ещё одно окно Командной строки** (не закрывая первое!)
2. Перейди в папку frontend:
```
cd M:\SEKS\АКТУАЛЬНОЕ\USNEE\usnee-frontend
```
3. Установи зависимости:
```
npm install
```
(Жди, пока скачается — может занять 2-5 минут)

4. Запусти frontend:
```
npm run dev
```

Если всё ок, ты увидишь:
```
  VITE v5.x.x  ready in XXX ms

  ➜  Local:   http://localhost:5173/
```

**Открой браузер и перейди по ссылке:** `http://localhost:5173`

Приложение должно открыться!

---

## Шаг 6. Запустить Telegram-бота (опционально)

Если у тебя есть BOT_TOKEN от @BotFather:

1. Открой **третье окно Командной строки**
2. Перейди в папку backend (уже с активированным venv):
```
cd M:\SEKS\АКТУАЛЬНОЕ\USNEE\backend
venv\Scripts\activate
```
3. Запусти бота:
```
python -m bot.main
```

---

## Как остановить

- В каждом окне командной строки нажми **Ctrl + C** и подтверди (Y + Enter)
- Чтобы снова запустить, повтори шаги 4.6 и 5.4 (для backend и frontend)

---

## Что делать, если что-то не работает

| Проблема | Решение |
|----------|---------|
| `python` не найден | Переустанови Python, обязательно поставь галочку "Add to PATH" |
| `node` не найден | Переустанови Node.js, перезагрузи компьютер |
| `pip install` падает | Проверь интернет, запусти `python -m pip install --upgrade pip`, потом повтори |
| `npm install` падает | Проверь интернет, удали папку `node_modules` и повтори `npm install` |
| Backend пишет ошибку БД | Удали файл `usnee.db` в папке `backend` — он пересоздастся |
| Порт 8000 занят | Закрой другие программы или запусти с другим портом: `uvicorn app.main:app --reload --port 8001` |
