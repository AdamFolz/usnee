# USNEE — Product Requirements Document
## Harm Reduction Tracker MVP

---

## Elevator Pitch

USNEE — безопасный трекер для людей, использующих ПАВ, с фокусом на harm reduction. Без осуждения, только факты. Приложение помогает отслеживать использование, анализировать паттерны, получать ачивки за безопасное поведение и находить ресурсы помощи.

---

## Problem Statement

Люди, использующие ПАВ, часто не ведут учёт своего потребления из-за стигмы и отсутствия безопасных инструментов. Это приводит к:
- Потере контроля над дозировкой
- Невозможности отследить опасные паттерны
- Отсутствию данных для врачей при передозировке
- Ощущению изоляции

---

## Target Audience

- Взрослые 18–40 лет, использующие инъекционные ПАВ
- Пользователи, практикующие harm reduction
- Люди, желающие получить контроль над своим использованием
- Пользователи Telegram Mini App

---

## Unique Selling Proposition

- **Без осуждения**: Нет морализаторства, только факты и инструменты
- **Геймификация**: Ачивки за безопасное поведение (использование фильтров, отслеживание интервалов)
- **Telegram Mini App**: Анонимность, никаких следов на устройстве
- **Экстренная помощь**: Один тап до спасительной информации

---

## Target Platforms

- Telegram Mini App (primary)
- Web (PWA fallback)
- Mobile-first responsive

## Features List

  - [ ] Подтверждение перед записью

  - [ ] Сегодня: количество инъекций
  - [ ] Время с последней
  - [ ] Средний интервал
  - [ ] Уровень и XP
  - [ ] График за неделю/месяц

- [ ] **История**
  - [ ] Список всех записей
  - [ ] Пагинация (cursor-based)
  - [ ] Отмена записи
  - [ ] Фильтр по дате

## UX/UI Considerations

### Главный экран
- [ ] **Состояние**: Карточка статистики сверху, триггеры по центру, форма записи
- [ ] **Анимация**: Появление ачивки — spring scale + rotate
- [ ] **Информационная архитектура**: Статистика → Триггер → Детали → Действие

### Экран записи
- [ ] **Состояния**: Неактивная кнопка (нет триггера) → Активная → Подтверждение → Успех
- [ ] **Прогрессивное раскрытие**: Триггер сначала, потом детали

### История
- [ ] **Состояние**: Пустая → Загрузка → Список → Бесконечный скролл
- [ ] **Визуальная иерархия**: Карточки с иконкой метода, датой, триггером

### Настройки
- [ ] **Состояние**: Список ссылок → Экран деталей
- [ ] **Навигация**: Back button на вложенных экранах

### Экстренная помощь
- [ ] **Состояние**: Кнопка всегда видна → Bottom sheet при клике
- [ ] **Визуальный акцент**: Красный цвет, пульсация
